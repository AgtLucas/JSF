Introdução - O que é o JSF?

Java Server Faces (JSF) é mais que um framework MVC para aplicações web em Java, se tornou um padrão e/ou especificação, faz parte do JEE e é mantido pela JCP (Java Community Process).

Possui um modelo de programação orientado a eventos, abstraindo os detalhes da manipulação dos eventos e organização dos componentes, permitindo que o programador se concentre na lógica da aplicação.

Atualmente, o JSF é considerado pela comunidade Java como a última palavra em termos de desenvolvimento de aplicações Web.

Histórico e versões

  • JSF 1.X usava JSP como sistema de template padrão.
  • A partir da versão 2.X, o sistema de template padrão passou a ser o Facelets.

Facelets

Criado por Jacob Hookom em 2005, é um sistema de template Web open source. Suporta todos os componentes do JSF e se concentra totalemnte na construção da árvore de componentes JSF, refletindo a view para uma aplicação JSF.

Baseado no padrão de projeto Composite, ou seja, seções de uma página ou mesmo as próprias páginas podem ser reutilizadas por outras páginas.

Facelets

Um Facelet pode ser definido como uma seção reutilizável de conteúdo ou uma sub-árvore de componentes em uma página JSF que pode ser por si mesma composta de outros Facelets. Neste sentido, um Facelet é um elemento de uma view composta que pode ser facilmente combinado com outros Facelets para construir uma view JSF que será apresentada ao usuário.

Como o JSF funciona?

  • JSF é baseado em um modelo de UI baseado em componentes, usando arquivos XML chamados de views ou Facelets views.
  • Os pedidos são processados pelo FacesServlet, que carrega o modelo da view adequado e constrói a ávorede de componentes, processa os eventos e apresenta a resposta para o cliente.

Como o JSF funciona?

  • O estado de componentes de UI e outros objetos de interesse de escopo, são salvos no final de cada pedido em um processo chamado stateSaving e restaurado na próxima criação dessa view.
  • Objetos e estados podem ser salvos tanto no lado do servidor, quanto no lado do cliente
